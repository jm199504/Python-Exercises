## 7.11 实现一个简单的问答系统

### 问题描述

提供以下的问题与答案库，实现文本匹配回答问题

| question               | answer                                               |
| ---------------------- | ---------------------------------------------------- |
| 中国总面积             | 中国的总面积约为960万平方公里（约为370万平方英里）。 |
| 陆地上最大的动物是什么 | 大象                                                 |
| 中国总人口             | 截至2022年，中国的人口约为14亿。                     |

### 输入输出示例

```
请问有什么问题需要解答？手机号几位
11位

请问有什么问题需要解答？中国面积
中国的总面积约为960万平方公里（约为370万平方英里）。

```
